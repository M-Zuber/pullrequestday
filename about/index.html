About International Pull Request Day

What is Pull Request Day?

Pull Request day is the last saturday of February, every year since 2016.

It's a day to encourage newcomers to Open Source contribution to make their first Pull Request, and for more experienced contributors to set aside some time to give something back.

How does it work?

We encourage people to make Pull Requests by publishing the Pull Request Pledge.

We coordinate events (like Pull Request Hack Day) where you can collaborate with others, or get a helping hand from people who have lots of experience working on Open Source projects.

We're still working this out though - we're actively looking for ideas partners to help us reach our goal of One Milllion Pull Requests in one day, so please do get in touch - either on Twitter or by emailing.

I think this is great. How can my organisation become a supporter ?

We're actively looking for organisations to help us, by promoting the event to their communities, friends, employees etc.

We haven't quite worked out the details yet, but we're also thinking about an institutional version of the pledge for organisations who employ developers - where they pledge that all of their engineers will make a Pull Request to an Open Source project on Pull Request Day. If you're interested and would like to discuss how that would work, drop us a line.

But what if you end up with one million terrible Pull Requests?

That's a valid concern. The last thing we want to do is create unhelpful noise or nuisance pull requests for project maintainers who are already overworked and volunteering their time. We think that it's possible to encourage Pull Requests that genuinely help though!

We're in the process of figuring out some guidelines for this - for how we can ensure that we help Open Source projects with new community contributions and inspire people to contribute more. If you have any ideas for how we can guide people in this regard, drop us a line.

Who's behind this ?

International Pull Request Day was founded by David & Sym in the Democracy Club Slack channel in 2015.
